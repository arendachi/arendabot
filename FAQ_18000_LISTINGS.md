## 🔧 **Проблемы и решения**

### ❌ **Проблема: "Объявления не найдены"**

**Причина:** Селекторы CSS устарели (сайт изменил HTML)

**Решение:**
1. Откройте сайт в браузере
2. Нажмите F12 (инспектор)
3. Найдите элемент с объявлением
4. Скопируйте класс/селектор
5. Обновите в скрипте (строка 100-110)

**Пример:**
```javascript
// Если раньше было:
const elements = document.querySelectorAll('a[href*="/listings/"]');

// А теперь селектор не работает, найдите правильный:
const elements = document.querySelectorAll('div[class*="property-card"] a');
```

---

### ❌ **Проблема: "Timeout - превышено время"**

**Причина:** Сайт медленный или блокирует парсер

**Решение:**
- Увеличьте таймауты в скрипте:
```javascript
const CONFIG = {
  timeout: 120000, // 120 сек вместо 60
  navigationTimeout: 90000, // 90 сек вместо 45
  delayBetweenPages: 5000, // 5 сек задержка между страницами
};
```

---

### ❌ **Проблема: "Out of memory" (нет памяти)**

**Причина:** GitHub Actions имеет ограниченную память

**Решение:**
- Уменьшите `maxListingsPerFile` чтобы сохранять чаще:
```javascript
maxListingsPerFile: 2000, // Вместо 5000
```

---

### ❌ **Проблема: Сайт блокирует парсер (403 Forbidden)**

**Причина:** Нет User-Agent

**Решение:** Уже добавлено в оптимизированной версии:
```javascript
await this.page.setUserAgent(
  'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36'
);
```

Если всё ещё блокирует - добавьте случайную задержку:
```javascript
const delay = Math.random() * 5000 + 2000; // 2-7 сек
await this.page.waitForTimeout(delay);
```
